## Scripts Debug


Build CLI with Development Mode
```sh
npm run build:dev
```

Build CLI with Production Mode
```sh
npm run build:prod
```

Watching CLI build
```sh
npm run cli:watch
```

Building CLI and Link to bin
```sh
npm run cli:build
```

Built CLI
```sh
npx cli
```
